### Sorted & lambda
s is [{"name": a, "house":b} {"name": c, "house":d} {"name": e, "house":f}]
we can sort by sorted
----
def get_home(a):
  return a["house"]

for l in sorted(s, key = get_home):
  print(f"{s["name"]} is in {s["house"]}")

----
or
----
for l in sorted(s, key = lambda x: x["house"]):
  print(f"{s["name"]} is in {s["house"]}")
----

we can give two keys in the sorted keys
for l in sorted(s, key = lambda x: (x["house"], x["name"]):
  print(f"{s["name"]} is in {s["house"]}")

one in ascending and other in descending
lambda x: (x["house"], -x["name"])

## regular expressions
inbuilt package re
import re

re.search("pattern", string, flag=0)

pattern = r".+"

. - any char
* - 0 or more times
+ - 1 or more times
? - 0 or 1 times (optional)
{m} - m times
{m, n} - range of m to n times

^.....$ - start and must be this pattern

[] - to allow only the specfied chars in the braces
eg: [',lo] allows only these chanracters

..* means . .* i.e 
. any char
.* any charachter 0 or more times

snippet 
str = rohith@gmail.com
re.search(r"^[a-z]+@[a-z]+\.com$", str)

one or more characters from a-z, @, one or more characters from a-z, .com
backslash used to tell . is not any character but period itself

we can use 
\w - any word inc all alphabets with both upper & lowercase and _
\W - opp to \w
\d - any digit
\D - opp to \d
\s - whitespace
\S - not whitespace

[ardc] allows a, r, d, c
[^cvb] except c, v, b

we can catch the sequence
name = "rohith malla"
