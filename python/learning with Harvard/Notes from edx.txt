## try except
try runs the code we want
except caches any errors and runs the code inside it
finally runs regardless of try and cache
else runs if except didn't cache the error

try:
   ...
except ValueError:
   ...
else:
   ...
finally:
   ...

use tuple in except, for multiple exception cache
eg: except (ValueError, ZeroDivisionError,...)

pass is an argument the just passes the block

raise Exception("hey this is an exception")
